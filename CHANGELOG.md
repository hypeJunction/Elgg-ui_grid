# Changelog

## 1.2.0 — Elgg 5.x

- Bumped minimum Elgg version to `^5.0` (PHP 8.2 required).
- No user-facing changes — plugin is pure CSS/views with no PHP logic.

## 1.1.0 — Elgg 4.x

- Migrated from Elgg 3.x to 4.x.
- Removed `manifest.xml`; `composer.json` is now the sole metadata source.
- Added `'plugin'` key to `elgg-plugin.php`.
- Bumped `composer/installers` to `^2.0`.

## 1.0.0 — Initial release

- Responsive CSS grid system for Elgg.
